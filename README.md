# TelegramBot счетчик

TelegramBot осуществляет подсчитываение сообщений всех участников чата и предоставляет на запрос обзор общее число сообщений за день или неделю.

## Используемые технологии/инструменты
##### Backend:
- Java 11
- Maven
- Spring Web
- Spring Data JPA
- Spring Boot
- GIT
- Stream API
- Lombok
- Hibernate

##### SQL:
- PostgreSQL
